# Telegram Digest Bot

Бот-помощник для составления ежедневного дайджеста новостей.

## Как это работает

1. В течение дня вы форвардите интересные новости в приватный/публичный канал-источник.
2. К некоторым из них можно добавлять хэштег (`#выборы`, `#коррупция` и т.п.) — такие новости группируются в тематические блоки в дайджесте.
3. По команде `/digest` бот собирает все накопленные новости, читает их полные тексты, генерирует пересказы через Claude API и публикует готовый дайджест в целевой канал.

## Версии

- **v0.1** (текущая) — сбор форвардов, ведение списка, базовые команды. Пересказы через Claude **не подключены**, `/digest` показывает только структуру будущего дайджеста.
- **v0.2** (следующая) — загрузка текстов статей и генерация дайджеста через Claude API.

## Установка (локально)

```cmd
cd c:\digestbot
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
copy .env.example .env
notepad .env
```

Заполните `.env` своими значениями (токен бота, API-ключ Anthropic).

## Запуск

```cmd
venv\Scripts\activate
python bot.py
```

## Команды бота

- `/start` — приветствие и статус
- `/list` — показать накопленный список
- `/tag N hashtag` — поставить тег
- `/tag N` — снять тег
- `/remove N` — удалить новость
- `/clear` — очистить весь список
- `/digest` — собрать дайджест

Все команды доступны только владельцу (`OWNER_ID` в `config.py`).
